<h2>Recipe</h2>
<section class="phase">
  <ul class="phase__ingredients">
    <li class="ingredient">
      <span class="ingredient__quantity">2</span> <span class="ingredient__unit">bags</span> <span class="ingredient__name">cranberries</span> <span class="ingredient__note">(fresh, washed)</span>
    </li>
    <li class="ingredient">
      <span class="ingredient__quantity">2</span> <span class="ingredient__unit">cups</span> <span class="ingredient__name">sugar</span>
    </li>
    <li class="ingredient">
      <span class="ingredient__quantity">1</span> <span class="ingredient__unit"><abbr title="tablespoon">Tbsp.</abbr></span> <span class="ingredient__name">orange zest</span>
    </li>
    <li class="ingredient">
      <span class="ingredient__quantity">1</span> <span class="ingredient__unit">cup</span> <span class="ingredient__name">orange juice</span>
    </li>
    <li class="ingredient">
      <span class="ingredient__quantity">1</span> <span class="ingredient__unit">cup</span> <span class="ingredient__name">water</span>
    </li>
  </ul>
  <ol class="phase__steps">
    <li>In a medium-sized pot, combine <span>cranberries</span>, <span>sugar</span>, <span>orange juice</span>, <span>water</span>, and <span>orange zest</span>.</li>
    <li>Bring to a boil then simmer for 10 minutes, stirring occasionally.</li>
    <li>Remove from heat, pour into a jar, and place in refrigerator to cool.</li>
  </ol>
</section>
